============
Contributing
============

mochada_kit welcomes contributions from the community. Please open an `issue <https://github.com/MaMMoS-project/mochada_kit/issues>`_ if you would like to report a bug, request a new feature or ask a question. 

If you would like to contribute code, such as fixing a bug or adding a new feature, please start the discussion by opening an `issue <https://github.com/MaMMoS-project/mochada_kit/issues>`_ to tell us what you'd like to do. Then follow the guides in this section while preparing your code and open a pull request when you're ready.
